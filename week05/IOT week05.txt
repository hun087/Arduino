IOT 좁은의미 : 유비쿼터스 환경 - 사물끼리 통신 주고받음

센싱 기술의 특징 - 해결되어야 한다.
1) 프로토콜의 이종성 -> 표준화로 문제해결함 (MQT)
2) 정보보안 문제 (개방형)
3) 자원 제약성 (비용) - 효율
배터리, CPU, 메모리
4) 이동성 - 인프라 구성 잘되어야 한다.

IOT 주요기술
근거리 : WPAN으로 지그비, 블루투스, NFC
무선이동 : 1G(아날로그) - 2G(음성 -> 디지털) :GSM, CDMA 등등 - ... 5G(3GPP 릴리즈)

차세대 기술 - 디바이스끼리 (6LOWPAN)


사물인터넷 개념
개체나 센서, 일상용품이 네트워크에 연결되고 정보처리 능력을 갖게 되어, 인간의 개입이 거의 없이 정보를 생성, 교환, 소비하여 정해진 기능을 수행하는 것

센서 - 물리적 환경의 변화를 감지하고 데이터 수집
엣지 디바이스 - 수집된 데이터를 처리하고 간단한 분석 수행 (디바이스간 데이터 취합)
게이트웨이 - 엣지 디바이스와 서버(클라우드)간의 중계 역할 (라즈베리파이, 미들웨어)
서버/클라우드 - 데이터를 저장, 분석 및 시각화
